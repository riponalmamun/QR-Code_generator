# QR-Code Generator

## 📌 Description
This is a simple Python script that generates a QR code from a given URL. You can use it to convert any web link into a scannable QR code image.

---

## 🔧 Prerequisites
- Python 3 installed on your system.

---

## 📦 Library Used
- `qrcode`
